# Tabletop Data Extraction

## EN  

This repository organizes the files and artifacts related to the production of the article “Digital Tabletop Interfaces: A Systematic Review of Technological Trends".

### Start here (main)
- If you came to this repository through the article and want the consolidated table, use [full_table_tex/accepted_articles.csv](full_table_tex/accepted_articles.csv) directly.

### Quick structure
- Table processing and chart generation pipeline: [data_processing/notebook.ipynb](data_processing/notebook.ipynb)
- Input data: [data_processing/input/](data_processing/input/)
- LaTeX article table generation: [full_table_tex/table_builder.ipynb](full_table_tex/table_builder.ipynb)
- PRISMA diagram: [prisma/prisma.html](prisma/prisma.html)
